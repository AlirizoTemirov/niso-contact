# Niso Bakery Cafe - Digital Business Card

A modern digital business card website built for Niso Bakery Cafe. The project provides customers with quick access to contact information, social media links, business hours, and location through a clean and mobile-friendly interface.

## Features

### Business Information

* Bakery logo and branding
* Business description
* Hero section

### Contact Options

* One-click phone call
* Telegram link
* Instagram link

### Business Details

* Working hours
* Location information
* Google Maps integration

### User Experience

* Responsive design
* Mobile-first layout
* Smooth animations
* Modern UI design

## Tech Stack

* Next.js
* React
* TypeScript
* Tailwind CSS
* React Icons

## Project Purpose

The goal of this project is to provide a simple and elegant online presence for a local bakery, allowing customers to quickly access important business information from any device.
